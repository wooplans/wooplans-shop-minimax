# Plan : Rendre le checkout fonctionnel

## Objectif
Rendre le checkout opérationnel en utilisant le Cloudflare Worker existant plutôt que la Supabase Edge Function.

---

## Étape 1 : Déployer le checkout-worker sur Cloudflare Workers

**Action** : Vérifier/déployer le worker avec les secrets nécessaires.

**Prérequis** (doivent être configurés dans GitHub Secrets ou localement) :
- `CLOUDFLARE_ACCOUNT_ID` - ID du compte Cloudflare
- `CLOUDFLARE_API_TOKEN` - Token API Cloudflare avec permissions Workers
- `CHARIOW_API_KEY` - Clé API Chariow

**Commandes locales** (si déploiement manuel) :
```bash
cd checkout-worker
wrangler login  # Authenticate with Cloudflare
echo "CHARIOW_API_KEY_value" | wrangler secret put CHARIOW_API_KEY
wrangler deploy --env production
```

**Résultat** : Obtenir l'URL publique du worker (ex: `https://wooplans-checkout.<user>.workers.dev`)
